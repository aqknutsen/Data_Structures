Includes Random Number Generator

Includes sorts: Bubble, Seleciton, Insertion, Quick Sort without median, Quick Sort with median, and Merge Sort

Will take an input file of random numbers, sort the numbers, and output a file with the sorts timed

Example Commands To Run:
./run -create -r Random.txt 10000 0 100000 - Create 10,000 random numbers ranging from 0 to 100,000 and store in Random.txt
./run sort -all Random.txt Sorted.txt - Run Sorts - Sort numbers in Random.txt output sorts in Sorted.txt
